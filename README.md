# ☄️ Komet

Tweetdeck for YouTube Live Streamers.

![Screenshot](https://github.com/holodata/komet/blob/master/.github/ss.png?raw=true)

## Komet?

コメ (/komé/) stands for comments/chats in Japanese, hence **Komet**.
